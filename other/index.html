動画情報取得 / Get Video Detail

  • GET https://www.nicovideo.jp/api/watch/v3_guest/{videoId}
  • GET https://www.nicovideo.jp/api/watch/v3/{videoId}

詳細な動画情報を取得します。
得られる情報はdiv#js-initial-watch-data -> data-api-dataと同じです。

PARAMETERS

PATH PARAMETERS

videoId* 動画ID(ex. sm9) string

QUERY PARAMETERS

actionTrackId* 任意のID(ex. wMuL0s1IZh_5459169147162) [a-zA-Z0-9]{10}_[0-9]{13}
skips (ex. harmful) string
noSideEffect (ex. false) boolean

HEADER PARAMETERS

x-Frontend-Id* PC版は固定値6 number
x-Frontend-Version* PC版は固定値0 number

COOKIE PARAMETERS

user-session ユーザーセッション(v3は必須です) string
